# CLMystery 日本語版（非公式）

[CLMystery](https://github.com/veltman/clmystery) に影響を受けて個人が作った、コマンドライン推理ゲーム集です。本家とは無関係の二次創作的なものです。

`grep`、`cat`、`less` などのコマンドを駆使して、ログファイルを調査し、事件の真相を突き止めてください。

## 作品一覧

| 作品 | 難易度 | 概要 |
|------|--------|------|
| [still-logged-in](./still-logged-in/) | ★★☆☆☆ | 1998年、大学研究棟で起きた転落死事件。サーバログから真犯人を特定せよ。 |
| [silent-daemon](./silent-daemon/) | ★★★☆☆ | 2003年、サーバ室で凍死した教授。cronとsudoのログが沈黙を破る。 |
| [closed-rooms](./closed-rooms/) | ★★★★☆ | 2007年、密室の研究室で教授が死亡。証言とログの矛盾をdiffで暴け。 |

## 遊び方

1. このリポジトリをクローンまたはダウンロード
2. 各作品のディレクトリに移動
3. READMEの指示に従って調査開始
4. **テキストエディタでログを開かないこと！** コマンドラインで調査してください
5. 答え合わせは各作品フォルダ内の `solution` ファイルを確認してください
