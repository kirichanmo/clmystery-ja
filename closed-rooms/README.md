## 閉じた部屋は、嘘をつかない

2007年の冬、私立大学の数理科学研究棟。
深夜、4階の研究室で一人の教授が亡くなりました。

部屋は内側から施錠されていた。
窓は閉まっていた。
心臓発作による自然死。
そう結論づけられようとしていました。

しかし、ある刑事は気づいたのです。
「証言が、合わない」と。

人は嘘をつく。しかし、ログは嘘をつかない。
そして、閉じた部屋たちは、静かに真実を語っている。

## 遊び方

1. `mystery/notes/incident.txt` を読んで事件を把握
2. `mystery/notes/residents.txt` で建物の住人を確認
3. `mystery/building/` 内のファイルで各部屋を訪問し、証言を集める
4. `mystery/testimonies/` で容疑者の詳細な証言を確認
5. `mystery/logs/` 内のログと証言を照合し、矛盾を見つけよ

## 部屋の訪問方法

各階のファイルは、行番号が部屋番号に対応しています。

```bash
# 407号室を訪問する（4階の7番目の部屋）
sed -n '7p' mystery/building/4F.txt

# または
head -7 mystery/building/4F.txt | tail -1
```

準備ができたら、`mystery/notes/incident.txt` から調査を開始してください。
